# 《科学技术普及法》知识测试应用

## 项目简介
本项目是一个基于 HTML、CSS 和 JavaScript 的交互式在线测试应用，旨在帮助用户学习和了解《中华人民共和国科学技术普及法》的重要内容。

## 功能特点
- 随机抽取10道科普法知识测试题
- 交互式答题界面，支持移动端和桌面端
- 实时答题进度追踪
- 答题结束后提供详细成绩报告
- 可分享测试成绩
- 响应式设计，兼容多种设备

## 技术栈
- HTML5
- CSS3
- JavaScript (原生)
- Bootstrap 5
